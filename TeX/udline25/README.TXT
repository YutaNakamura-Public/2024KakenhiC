【ソフト紹介】
TeX で使える下線の種類を増やそう，ということと，改行を可能にしよう
という2つの目的をもって作ったパッケージです。
同種のパッケージもいろいろ出ていますので，
今更という感じがしないでもありませんが。(^_^;;

《特徴》
・途中改行可能な下線や飾り下線を引ける
・下線・飾り下線までの距離を調節できる
・下線の太さを変えられる
・飾り線がデフォルトで8種用意されている(ver 2.50 から1種増えました)
・しかも自分で新しい飾り下線を定義することもできる
・縦書き・横書き・和文・英文対応

《使い方》
パッケージファイルとして必要なものは udline.sty だけです。
これを適当なディレクトリ(たとえば \ptex\share\texmf\tex\
の下に MyStyle\ などの名前でディレクトリを作って)に保存して下さい。
これで udline.sty が使えるようになります。
具体的な使用方法については，添付のマニュアル (manual.tex) と
サンプル (sampleul.tex) をタイプセットして御覧下さい。

《転載条件》
このマクロは自由に改造して使っていただいて結構ですが，
それを「転載」するのはおやめ下さい。
転載の際にはファイル構成などに手を加えていないものでお願いします。
(大きく改造したものは「オリジナル」とは別物 ==「改造した人の著作物」
として，別個に登録して下さい。)
それ以外の点に関しては，基本的に転載自由です。
